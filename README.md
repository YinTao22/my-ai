# my-ai

一个简单ai聊天机器人。其中，可以修改提示词，能变成你的百变Ai机器人呢。当然有点缺点就是使用R1模型很慢，大概需要30多秒之后才能得到回复!😭 :sob:


# 项目部署
## 克隆此项目
```text
git clone https://github.com/YinTao22/my-ai.git
```
## 安装所需的库
1： requests用于发送 HTTP 请求，需要安装。
```text
pip install requests
```
2 ：rich：
用于增强控制台输出，需要安装。
```text
pip install rich
```
